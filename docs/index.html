What it does

Backups you set once and stop thinking about.

Point it at the folders you'd hate to lose, give each one a schedule, and SyncSentinel keeps a mirror in sync — quietly, from the tray.

Scheduled mirroring

Each job runs on its own interval, with robocopy as the engine. The clock is anchored to the last finish, so a slow run never piles up — and a job never overlaps itself.

one global queue · run now jumps it

Reusable exclusion sets

Define bin, obj, node_modules once and attach the set to any job. Folder names feed robocopy /XD; file patterns feed /XF.

composable · union of every attached set

Live status & history

Watch each run stream live, then review what happened: status, files copied / skipped / extra / failed, exit code, and the full per-run log.

retention prunes by count or age

How it works

Four steps, then it runs itself.

Add a job

Point at a source folder and a destination. Browse with the native Windows picker, or paste a path.

Tune the sweep

Set the interval, attach exclusion sets, and adjust the robocopy flags — or keep the sensible defaults.

It stands watch

On each due tick it mirrors with robocopy. One run at a time; a missing source is skipped, not failed.

Review

See the live log while it runs, then the status dot, counts, and full log for every past run in history.

Details

The fine print, done right.

/XD /XF Exclusion sets

Reusable folder and file exclusions, composed into each run.

/MIR Flag editor

Pick robocopy flags from a described catalog, or add your own.

History

Per-run counts, exit codes, and logs — pruned by your retention rules.

Tray & autostart

Lives in the system tray; starts on login, per-user, no admin.

Single .exe

Self-contained, ~62 MB. A no-admin installer is offered too.

Loopback-only

The UI is served on 127.0.0.1 and never exposed to the network.

Preconditions

A run is skipped — not failed — when its source or drive is missing.

Live updates

Status and log lines stream to the UI as the run happens.

Get it

Put SyncSentinel on watch.latest

Windows 10/11, 64-bit. The installer needs no admin rights; the portable build needs no install at all.
